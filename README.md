# Задание DiveSea

### После клонирования
```
npm install
```

### Запуск в режиме разработки
```
npm run dev
```
(доступно на [http://localhost:5173](http://localhost:5173))

### Сборка и запуск production
```
npm run build
npm run preview
```
(доступно на [http://localhost:5173](http://localhost:4173))
